# Blend-Bite-Frontend-Development

Welcome to the **Blend & Bite** frontend project! This is a modern, responsive multi-page website designed for a restaurant/cafe setting, focusing on a clean user interface and smooth navigation.

**Project Overview:**

Blend & Bite is a web-based platform that allows users to explore a culinary world. From viewing our curated menu to browsing a gallery of our finest dishes, this project demonstrates core frontend development principles using HTML5 and CSS3,Bootstrap and JavaScript.

**Project Structure:**

The repository contains the following key pages:

 **HomePage.html**: The landing page featuring the brand identity and primary call-to-actions.
 
 **Menu.html**: A detailed showcase of food and beverage offerings.
 
 **About.html**: Information about the brand's history, mission, and values.
 
 **Gallery.html**: A visual display of the restaurant's ambiance and dishes.
 
 **Contact.html**: A functional layout for users to get in touch or make inquiries.

**Tech Stack:**

 **HTML5**: For semantic page structure.
 
 **Bootstrap 5**: For responsive layouts, components, and rapid styling.
 
 **JavaScript (JS)**: For dynamic UI interactions and form handling.
 
 **CSS3**: For custom styling and branding.
